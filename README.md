
# 保利威实时音视频套件 PRTC SDK

## 介绍

实时音视频（Polyv Real-Time Communication，简称PRTC），依托于Polyv强大的覆盖能力，可以实现用户的就近接入，提供网络低延迟、低丢包率的音视频通信。
PRTC SDK支持Web、Android、iOS、macOS、Windows、Electron多种客户端的集成接入。通过集成PRTC SDK，能快速构建多人音视频通话、一对多或多对多的实时互动直播。
